# Library Management System

This project is a Library Management System built to handle core library operations such as managing books, users, and borrowing records. The primary goal of the project is to design a clean, scalable backend with a simple and usable interface on top of it.

The system is structured to follow good software engineering practices, with clear separation of concerns and maintainable code organization.

---

## Features

- Manage library books (add, update, remove, search)
- User authentication and role-based access control
- Book issue and return tracking
- Due date management and overdue fine calculation
- Search and filtering functionality
- Admin-level controls for managing library data

---
